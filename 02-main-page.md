# 02. Main Page — 메인 = 판매페이지 (장문 세일즈 랜딩)

세온애드 웹사이트의 **메인 페이지(`/`)는 곧 판매페이지**다.
방문자가 스크롤을 따라 내려가며 문제 인식 → 해결책 → 증거 → 가격 → 신청까지 가는 **장문 세일즈 구조**로 설계한다.

- **주 전환**: 사전 무료 강의 신청 (`/free`)
- **2차 전환**: 본 강의 수강 신청 (`/apply`) → **법인계좌 이체** (주식회사 티오엠)
- **참조 토큰**: `01-foundation.md`

> **결제 방식 확정 (v3)**: PG사 연동 없이 **법인계좌 이체 단독**, **일시불 70만 원** 단일 옵션으로 운영한다. 할부·분할 납부는 제공하지 않는다. 신청 → 대표자가 카테고리 중복 확인 → 입금 안내 (카카오톡 채널 수동 응대) → 입금 확인 → 기수 확정 → 카카오톡 단톡방 초대의 수동 플로우이며, 이는 "소수정예·경쟁사 보호"라는 브랜드 포지션과 일관된 운영 방식이다. 05-pricing-apply.md와 06-faq.md는 이 결정을 그대로 따른다.

## 1. 공통 — 전역 요소

### 1-1. 헤더 (sticky)

- 배경: `--color-navy-950` (최초), 스크롤 시 `--color-navy-900` + 하단 1px `--color-gold-500` 라인
- 높이: 데스크톱 72px, 모바일 56px
- 좌측: SEONAD 로고 (골드 라인아트) + 태그라인 "스마트스토어 네이버광고 소수정예 수업" (md 이상에서만 노출)
- 중앙: 메뉴 링크 — 커리큘럼 / 강사 / 가격 / FAQ / 블로그
- 우측: **"사전 무료 강의 신청"** Primary CTA
- 좌측 상단 모집 배지(옵션): `"현재 기수 4/6명"` — `--color-gold-400` 필 배지, 없을 때 숨김
- 모바일: 로고 + "무료 강의" 축약 CTA + 햄버거, 오버레이 메뉴는 `--z-overlay`

### 1-2. 모바일 하단 플로팅 CTA

- 히어로 통과 지점(스크롤 50%)부터 노출
- 전체 폭 90% Primary 버튼, `--shadow-gold`
- 기본 문구: "사전 무료 강의 신청 →"
- 가격 섹션(섹션 11) 진입 시 문구 교체: "수강 신청하기 →" + `/apply` 이동

### 1-3. 섹션 리듬 원칙

- 다크 섹션(navy)과 라이트 섹션(paper-soft/cream)을 **3~5블록 단위로 교차**
- 각 섹션 상하 패딩: 데스크톱 `--space-10`, 모바일 `--space-8`
- 섹션 간 전환 문구(transition copy)를 짧게 배치해 스크롤 이어짐 유도

## 19. 바이브 코딩 작업 체크리스트

1. 01-foundation 토큰을 루트 CSS에 주입
2. 섹션 1~15 순서로 컴포넌트 분리 생성
3. `{{CURRENT_COUNT}}`, `{{NEXT_START_DATE}}` 변수는 별도 설정 파일로 분리
4. 헤더·푸터·플로팅 CTA는 전역 레이아웃에 배치
5. Schema Markup 3종은 `<head>` 주입 방식으로 별도 컴포넌트화
6. 모바일 first 레이아웃 후 `--bp-lg` 기점 데스크톱 스타일 적용
7. 히어로·섹션 9(강사)·섹션 15(최종 CTA)에 스크롤 페이드(--duration-scroll) 적용, 그 외는 정적 렌더
8. 이미지·목업은 v1 런칭 시 플레이스홀더(골드 라인 와이어프레임) 사용 → 런칭 후 실제 촬영본으로 교체
