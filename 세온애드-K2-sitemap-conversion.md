- 신청자 전원에게 이메일 + 카톡 리마인드
- 무료 강의 시청 후 결제 유도 링크 전달

**2순위 — 본 강의 결제 (70만 원 전환)**

- **일시불 70만 원**만 받음 (카드 할부·분할 납부 없음)
- 입금 확인 완료 시 카톡 단톡방 초대 안내 (신청 순번에 따른 기수 배정)

**3순위 — 블로그·유튜브 구독 (장기 리드)**
- 블로그 글 하단 뉴스레터 구독
- 유튜브 구독 버튼 노출
- 추후 다음 기수 안내 대상

## 메인 판매페이지 섹션 순서 (초안)

메인이 곧 판매페이지이므로, **스크롤 따라가며 결제로 가는 세일즈 구조**로 설계합니다.

1. **히어로** — "한 기수 6명. 4주 뒤, 내 손으로 광고 계정을 돌립니다."
2. **문제 제기** — 초보 셀러가 네이버광고에서 겪는 현실 (돈만 나가고 주문은 없다)
3. **세온애드 소개** — 무엇이 다른가 (6명 + 실계정 공개 + 직접 운영)
4. **커리큘럼 요약** — 4주 구조 시각화 (VOD 2주 + 줌 2주 + 애프터케어 3개월)
5. **VOD 7강 + 부록 미리보기** — 강의별 목표·시간·화면 따라하기 마크
6. **줌 2회 상세** — 1주차/2주차 진행 방식, 실계정 공개의 의미
7. **애프터케어 안내** — 단톡방 3개월 + 점검 줌 1회 + 경쟁사 보호 정책
8. **제공 자료** — 워크북 + 체크리스트 3종 + AI 도우미 2종 시각화
9. **강사 소개** — 세온비즈 경력, 30개 업체 코칭, 4개 업체 실적 사전 무료 강의 공개 안내
10. **경쟁 비교 테이블** — 대형 플랫폼 / 유튜브 / 대행사 / 안 하기 / 세온애드
11. **가격 안내** — 70만 원 포함 내역 시각화, 일시불·법인계좌 이체, 법인 세금계산서 수기 발행
12. **환불 정책** — 투명한 조건 (강의 전 100% / VOD 1강 전 100% / 이후 불가 + 다음 기수 재수강)
13. **FAQ 하이라이트** — 6~8개 핵심 질문만 추출, 나머지는 FAQ 페이지
14. **최종 CTA** — "사전 무료 강의 신청" + "지금 기수 몇 명 남았는지 표시"

### 스크롤 유도 장치
- 섹션 간 **전환 문구** 배치 (예: "그래서, 어떻게 배우는가?" → 커리큘럼 섹션)
- **숫자 강조 섹션** (6명 / 4주 / 3개월 / 70만 원)을 반복적으로 노출
- **실시간 모집 현황** (4/6명)은 히어로와 최종 CTA에 노출

## 방문자 유형별 동선

### 페르소나 1 (초행형, 1차 타깃) — **전환까지 예상 방문수 2~3회**
- 1차 방문: 검색·광고 유입 → 메인 스크롤 → 이탈 (70%)
- 2차 방문: 리마케팅 광고 → 사전 무료 강의 신청 (30%)
- 3차 방문: 무료 강의 시청 후 결제 (10~15%)

### 페르소나 2 (실패형) — **전환까지 예상 방문수 1~2회**
- 1차 방문: 사전 무료 강의 신청 (문제 진단 욕구)
- 2차 방문: 무료 강의 시청 후 결제

### 페르소나 3 (직영 전환형) — **전환까지 예상 방문수 2~4회**
- 여러 번 방문해서 FAQ·강사·커리큘럼 정독
- "직접 해봐야 대행도 잘 맡긴다" 메시지에 반응
- 결제 시점이 느리지만 결제하면 후회 적음

### 페르소나 4 (가치형) — **전환까지 예상 방문수 1회**
- 히어로에서 "6명" 보자마자 경쟁사 보호 정책 확인
- 강사 페이지 한 번 확인 후 빠르게 결제
- **가장 결제 의사결정이 빠른 페르소나**

### 각 페르소나별 이탈 방지 장치
- 페르소나 1: 메인 중반부에 "완전 초보 비즈센터 가입부터" 섹션 강조
- 페르소나 2: "왜 망했는지 그룹에서 진단" 섹션 배치
- 페르소나 3: "직접 경험한 대표만 주도적으로 맡긴다" 프레임
- 페르소나 4: 히어로 바로 아래 "6명 · 경쟁사 보호 · 실계정" 3요소 배지

1. 이름 (실명)
2. 연락처 (카카오톡 ID 또는 휴대폰 번호)

### 선택 필드 (전환율 높이는 용도)
- 현재 월 광고비 규모 (없음 / 10~30만 / 30~50만 / 50만 이상)
- 광고 경험 (없음 / 파워링크만 / 쇼핑검색 시도 / 기타)

### 신청 후 프로세스
1. 즉시: 이메일·카톡으로 줌 링크 + 일정 안내
2. 강의 1일 전: 리마인드 메시지
3. 강의 당일: 시작 2시간 전 알림
4. 강의 종료 직후: 본 강의 신청 링크 + 할인 혜택 (있다면)

6. 강의 7일 후: 마지막 리마인드

## 수강 신청 프로세스 (법인계좌 이체)

### 신청 흐름
1. `/apply` 페이지에서 **신청 폼 작성** + **약관 동의** (수강 약관 + 환불 정책 + 경쟁사 보호 정책 + 계정 공개 동의)
2. 신청 제출 → 자동 안내 이메일 발송 (법인계좌 정보 포함)
3. 운영자가 신청자 카테고리 확인 + 카카오톡 채널로 **수동 안내**
   - 법인계좌: 주식회사 티오엠 명의
   - 입금 금액: 70만 원 (일시불)
   - 입금 마감: 안내 후 48시간
4. 수강생 입금 → 운영자가 입금 확인 → 기수 배정 확정
5. **경쟁사 보호 정책 검토**: 카테고리 겹치면 입금 전 안내 (선 신청자 우선 원칙)
6. 확정 시 **카카오톡 단톡방 초대**, 기수 시작일 안내

### 결제 수단
- **법인계좌 이체 단독** (주식회사 티오엠 명의)
- **일시불 70만 원만** 받음
- 신용카드 할부·분할 납부·PG 결제 없음
- "자동 결제가 아니라, 대표자가 직접 배정을 확인하는 소수정예 운영 방식"으로 포지셔닝

### 법인 세금계산서
- 신청 폼에 "세금계산서 발행 희망" 체크박스
- 사업자번호·법인명·담당자 이메일 입력 필드
- **주식회사 티오엠** 명의로 **수기 발행** (입금 확인 완료일 기준)
- FAQ와 신청 페이지 모두에 "법인·개인사업자 세금계산서 수기 발행 가능" 명시

### 신청 이탈·미입금 대응
- 신청 후 48시간 미입금 시 → 대기열로 자동 이월 안내 (카톡 채널)
- `/apply` 페이지 이탈자 대상 리타깃팅 광고
- 카톡 채널 친구라면 카톡 리마인드 (수동, 1회만 발송 — 과도한 독촉 방지)

## 경쟁사 보호 정책 — 신청 프로세스 상세

### 처리 순서
1. 신청 폼 제출 → 신청자 정보 DB 등록 (이름, 카테고리, 신청 일시)
2. 운영자가 현재 기수 신청자 목록과 카테고리 대조 (**입금 안내 전에** 1차 검토)
3. **카테고리 겹침 발생 시**:
   - 선 신청자 → 현재 기수 확정 (입금 안내 진행)
   - 후 신청자 → 카카오톡으로 안내: "다음 기수 배정 예정 (예상 시작일 X월 X일) / 입금 없이 대기 순번 X번 / 본인이 취소 요청 시 즉시 이월 해제"
4. 후 신청자가 **대기 선택 시** 다음 기수 최우선 배정
5. 후 신청자가 **취소 선택 시** 입금 전이므로 별도 환불 없이 이월 해제
6. 이미 입금 후 카테고리 겹침이 뒤늦게 확인된 경우 → 3영업일 내 100% 환불 또는 이월 선택

### 판매페이지 표현
