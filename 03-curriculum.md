# 03. Curriculum — 커리큘럼 상세 페이지 (`/curriculum`)

메인 판매페이지 섹션 5~6에서 요약만 본 방문자가 **"진짜로 4주간 뭘 배우는지"** 상세 확인하는 페이지.
결제 직전의 마지막 의심(초행형 페르소나 중심)을 해소하고 다시 메인 또는 신청 페이지로 되돌려보내는 역할.

- **역할**: 2차 전환 전 마지막 검증 페이지
- **주 CTA**: 사전 무료 강의 신청 (`/free`)
- **보조 CTA**: 수강 신청하기 (`/apply`) · 가격 페이지 (`/pricing`)
- **참조 토큰**: `01-foundation.md`

- 헤더·푸터·모바일 플로팅 CTA는 `02-main-page.md`의 전역 설정을 그대로 사용

### 4-1. 배경·목적
- 배경: `--color-paper` (흰색)
- 목적: 1주차 VOD 2강 풀 상세 공개 — 초행형 페르소나 안심의 핵심 지점

### 4-2. 섹션 헤더

**주차 라벨** (골드 배지, `--text-caption`):
```
WEEK 1 · VOD 60분 (2강)
```

**제목** (`--text-h2`, `--color-ink-900`):
```
PART 1 — 광고를 켜기 전에 알아야 할 것.
```

**서브** (`--text-lead`, `--color-ink-500`):
```
광고를 "시작할 준비가 된 상태"를 만드는 구간.
이게 안 된 상태로 광고를 켜면 4주차가 와도 수익이 안 납니다.
```

### 4-3. 1강 카드

**카드 헤더**
```
[배지: 1강 · 25분 · PART 1]
스마트스토어 셀러의 네이버광고 지도
```

**강의 목표** (`--text-body`, 박스 안):
```
💡 이 강의가 끝나면:
4가지 광고 상품(파워링크·쇼핑검색·브랜드검색·GFA) 중
내 상황에 필요한 것을 판단할 수 있습니다.
```

**다루는 내용** (불릿 리스트):
- 네이버 광고 생태계 지도 — 파워링크·쇼핑검색·브랜드검색·GFA의 위치와 역할
- 각 상품이 누구에게, 언제 맞는지
- 이 강의에서 다룰 3가지 상품의 우선순위 — 왜 쇼핑검색이 메인인가
- 광고 시작 전 체크리스트 — 상세페이지, 리뷰, 가격, 썸네일
- 학습 확인 — 내 스토어가 광고를 켜기에 준비된 상태인지 자가 진단

### 4-4. 2강 카드 (화면 따라하기 ★ 마크)

**카드 헤더** (골드 테두리 강조, `--color-gold-500` 2px)

**강의 목표**:

💡 이 강의가 끝나면:
네이버 비즈센터 가입부터 전환추적 설치까지 완료 상태가 됩니다.

**다루는 내용**:
- 네이버 비즈센터 가입 → 광고주 가입 → 비즈머니 충전

- 첫 화면 구조 이해 — 계정 → 캠페인 → 그룹 → 소재/키워드

### 5-2. 섹션 헤더

```

**제목**:

```

**서브**:
```
스마트스토어 셀러에게 1순위인 쇼핑검색부터,
파워링크·브랜드검색까지 강의 시간과 우선순위를 배분했습니다.
```

### 5-3. PART 2 서브헤더 (`--text-h3`, `--color-gold-600`)
```
PART 2 — 쇼핑검색광고 (메인 파트 · 85분)
```

### 5-4. 3강 카드

```
[배지: 3강 · 25분 · PART 2]
쇼핑검색광고 전략 설계
```

**목표**: 내 상품 중 어떤 것을 먼저 광고할지, 그룹을 어떻게 나눌지 결정할 수 있습니다.

**다루는 내용**:
- 쇼핑검색광고가 스마트스토어 셀러에게 1순위인 이유
- 소재 단위 = 상품 단위라는 구조 이해
- 어떤 상품을 먼저 광고할지 — 베스트·마진·전환율 3축
- 캠페인·그룹 구조 설계 — 카테고리·성별·계절·가격대 분리 원칙
- **워크북 연동**: PART 2 쇼핑검색 설계 워크시트 작성

### 5-5. 4강 카드 (화면 따라하기 ★ 마크)

```
[배지: 4강 · 35분 · PART 2 · ▶ 화면 따라하기]
쇼핑검색광고 세팅 실습
```

**목표**: 첫 쇼핑검색광고 캠페인 1개를 실제로 발행할 수 있습니다.

**다루는 내용**:
- 캠페인 생성 → 그룹 생성 → 상품 연결 → 입찰가 설정
- **입찰가 결정 공식**: 마진 × 전환율 × 목표 ROAS
- 확장 소재, 쇼핑 이미지, 소재 승인 규정
- 제외 키워드, 제외 지역, 노출 시간 설정
- **실습 결과**: 첫 캠페인 1개 실제 발행 (줌 1주차 과제의 준비 상태)

### 5-6. 5강 카드

```
[배지: 5강 · 25분 · PART 2]
쇼핑검색광고 최적화 사이클
```

**목표**: 광고를 켠 뒤 언제 입찰을 조정하고, 언제 끄는지 판단할 수 있습니다.

**다루는 내용**:
- 성과 지표 4개 — 노출·클릭·전환·ROAS
- 일간·주간·월간 봐야 할 숫자
- 입찰가 조정 규칙 — 언제 올리고 언제 내리는가
- "광고 끄기" 의사결정 기준 — 손실 확정 라인
- **체크리스트 연동**: 일간·주간 체크리스트 사용법

### 5-8. 6강 카드 (화면 따라하기 ★ 마크)

[배지: 6강 · 40분 · PART 3 · ▶ 화면 따라하기]

**목표**: 내 상품에 맞는 키워드 20개를 추출하고 첫 파워링크 캠페인을 발행할 수 있습니다.

**다루는 내용**:

- 검색량·경쟁도·CPC 판단 기준 (네이버 키워드 도구 활용)

- 품질지수가 실제로 돈을 아껴주는 원리
- 캠페인·그룹 생성 실습
- **실습 결과**: 핵심 키워드 20개 + 첫 파워링크 캠페인 발행

### 5-9. 7강 카드

```
[배지: 7강 · 30분 · PART 3]

```

**목표**: 브랜드검색을 언제 써야 할지 판단하고, 전체 광고비를 어떻게 분배할지 결정할 수 있습니다.

**다루는 내용**:
- 브랜드검색 — 언제 필요한가, 비용 구조(정액제), 소재 유형(라이트/프리미엄)
- 본 강의에서 "당장 필요하지 않다"는 판단 기준
- 광고비 예산 분배 원칙 — 쇼핑검색 vs 파워링크
- 어떤 지표가 나빠지면 행동해야 하는가
- 쇼핑검색·파워링크 조합 시너지

- 일·주·월 운영 사이클 설계
- 어떤 지표가 나빠지면 행동하는가
- 월 단위 광고비 계획 수립
- 신규 상품 추가 시 광고 확장 원칙
