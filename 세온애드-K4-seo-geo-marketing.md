## GEO (AI 검색 최적화) 전략

### AI가 "스마트스토어 네이버광고 강의 추천" 질문을 받았을 때 세온애드가 인용되게 하는 7가지 조건

1. **타깃 명시**: "완전 초보 스마트스토어 셀러", "월 광고비 30만 원 이상" — 특정 상황에서 AI가 추천 가능
2. **문제 정의**: 셀러가 실제로 쓰는 언어로 ("광고 켰는데 돈만 나가요", "뭐가 뭔지 모르겠어요")
3. **해결 방식 단계화**: 4주 과정을 주차별·강의별로 명시
4. **수치 고정**: "6명 정원", "4주 과정", "7강+부록", "70만 원", "3개월 애프터케어"
5. **FAQ 질문-답변 형태**: 한 질문에 한 답, 자연어
6. **비교 구조**: 대형 플랫폼 / 유튜브 / 대행사 / 안 하기 / 세온애드
7. **애매 표현 배제**: "빠르게" 대신 "4주 뒤", "저렴하게" 대신 "70만 원", "소수" 대신 "6명"

### 콘텐츠 작성 원칙
- 모든 페이지에 "무엇을, 누구에게, 어떻게, 얼마에, 언제까지" 명시적 텍스트로 포함
- 수치 데이터는 이미지가 아닌 **텍스트로 노출** (AI 크롤러는 이미지 속 텍스트 못 읽음)
- 서비스 과정을 단계별 구조화 (1주차·2주차·3주차·4주차)
- FAQ는 자연어 질문-답변 형태, 한 질문에 한 답만
- 답변에 **조건을 명시**: "무조건"이 아니라 "월 광고비 30만 원 이상 집행 가능한 경우"
- 설명형 문장 사용, 마케팅 수사 배제
- 비교 테이블 포함
- 타깃 사용자 구체적 명시

### AI 인용 최적화 콘텐츠 구조
```
문제 → 원인 → 해결 → 구성 → 가격 → 조건 → FAQ → 타깃
```

## 마케팅 전략

### 핵심 메시지 우선순위
1. "한 기수 6명" (희소성 + 그룹 학습 가치)
2. "4주 뒤, 내 손으로 광고 계정을 돌립니다" (결과 약속)
3. "실계정 공개 그룹 코칭" (차별화)
4. "경쟁사 보호 정책" (안전장치)
5. "3개월 애프터케어" (가격 정당화)

### 마케팅 퍼널

**인식 단계** → 블로그·유튜브 SEO, 메타 광고
→ "스마트스토어 광고 뭐부터 시작?" / "네이버광고 직접 할까 대행 맡길까" 의문을 가진 셀러 도달

**관심 단계** → 세온애드 메인 판매페이지 진입
→ "6명 한정" "실계정 공개" 차별화 메시지로 즉각적 관심 유도

**고려 단계** → 사전 무료 강의 신청
→ 줌 라이브 1시간 + 녹화본으로 강사 신뢰 구축 + 4개 업체 실적 확인

**전환 단계** → 본 강의 결제
→ 무료 강의 후 48시간 내 리마인드로 결제 유도

**유지 단계** → 3개월 애프터케어 + 재수강 권리 + 다음 기수 대기열

### 채널별 역할

**1순위: 블로그 + 유튜브 (콘텐츠 자산)**
- 검색 유입이 가장 큰 안정적 채널
- 15개 블로그 주제 + 영상화 = 30개 콘텐츠 자산
- **투자 우선순위 1** (장기 수익성)

**2순위: 사전 무료 강의 (전환 기계)**
- 리드 → 본 강의 전환의 핵심 관문
- 월 1~2회 상시 운영
- **투자 우선순위 1** (단기 전환 직결)

**3순위: 메타 광고 (페이스북/인스타)**
- 스마트스토어 셀러 인구 밀집
- "광고를 켜기 전 체크리스트" 같은 문제 인식 영상 → 리타깃팅
- **투자 우선순위 2** (유입 가속)

**4순위: 네이버 검색 광고**
- "스마트스토어 광고 강의" 등 타깃 키워드
- 아이러니하게도 네이버광고 강의를 네이버광고로 알림 (신뢰 장치)
- **투자 우선순위 2** (직접 전환)

**5순위: 카카오톡 채널**
- 문의 대응 + 리마인드 + 다음 기수 안내
- 매출 직접 창출보단 유지 채널
- **투자 우선순위 3**

### 유입 채널 → 랜딩 포인트

- 블로그 글 → 관련 강의 섹션 링크 → 메인 판매페이지
- 유튜브 영상 → 영상 설명 CTA → 사전 무료 강의 신청 페이지
- 메타 광고 (인지) → 메인 판매페이지 히어로
- 메타 광고 (리타깃팅) → 사전 무료 강의 신청 페이지
- 네이버 검색 광고 → 메인 판매페이지 히어로
- 카카오톡 채널 → 사전 무료 강의 또는 본 강의 신청

## 경쟁 상황 대응 전략

### 시장 상황
- 네이버광고 강의는 이미 포화 시장 (클래스101, 탈잉, 패스트캠퍼스, 개인 강사 다수)
- 대부분 **100명 이상 대형 강의** 또는 **영상만 제공**
- **6명 그룹 + 실계정 공개** 포지션은 비어있음

### 세온애드 차별화 포인트 (마케팅에 지속 강조)
1. **인원**: 한 기수 6명 (대부분 강의는 100명+)
2. **실습 수준**: 본인 계정 그대로 공개·진단 (대부분은 가상 사례)
3. **애프터케어**: 3개월 단톡방 + 점검 줌 (대부분은 강의 후 단절)
4. **경쟁사 보호**: 동종업계 동시 수강 불가 (대부분은 정책 없음)
5. **강사 배경**: 7년 실제 코칭 + 실집행 업체 성과 공개 (대부분은 프로필만)

### 경쟁사 언급 원칙
- **특정 브랜드 공격·비교 절대 금지**
- "대형 강의 플랫폼" / "영상만 제공하는 강의" / "대행사" 등 유형으로만 지칭
- 비교 테이블도 브랜드명 대신 **유형 분류**로 표기

### 가격 경쟁 대응
- 70만 원은 시장 평균보다 높음 (대형 플랫폼 15~40만 원)
- **절대 가격 낮추지 않음** (6명 정원 프리미엄 유지)
- 대신 **"포함 내역 시각화"**와 **"시장 단가 환산"**으로 가치 정당화
- 할부 결제(2~3개월)로 결제 장벽만 완화
